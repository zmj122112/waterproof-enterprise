## 技术要点

1. 使用 Tailwind CSS 类名或内联样式实现设计要求
2. 确保颜色值严格使用品牌红 #E60012 和品牌金 #F59E0B
3. 注意元素的层级关系，使用 z-index 确保正确显示
4. 使用 Flex 和 Grid 布局实现响应式设计
5. 注意阴影和圆角的使用，增强视觉层次感
