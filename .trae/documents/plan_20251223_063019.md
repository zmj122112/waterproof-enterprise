# 业务应用层（中台）页面开发计划

## 1. 现有系统分析
当前项目已包含四个主要平台：
- C端小程序 (c-mini)
- B端门户 (b-portal)
- 管理系统 (admin-system)
- 师傅端 (master-app)

## 2. 中台功能规划
根据用户需求，我将为中台添加以下功能模块：

### 2.1 核心业务管理
- **订单管理**：增强现有工单管理功能，添加订单详情、流程跟踪
- **合同管理**：完善合同创建、审核、执行流程
- **项目管理**：项目全生命周期管理
- **财务管理**：收支管理、对账、发票管理

### 2.2 资源管理
- **设备管理**：设备台账、维护记录
- **知识库管理**：技术文档、案例库
- **供应商管理**：供应商信息、合作记录

### 2.3 数据中心
- **数据统计**：关键业务指标统计
- **报表中心**：各种业务报表生成
- **日志管理**：系统操作日志、业务日志

### 2.4 系统配置
- **角色权限管理**：精细化权限控制
- **参数配置**：系统参数设置
- **通知管理**：消息模板、推送设置

## 3. 页面创建计划

### 3.1 核心业务管理页面
- `admin-system/OrderDetail.vue`：订单详情页
- `admin-system/ContractManagement.vue`：合同管理列表页
- `admin-system/ContractDetail.vue`：合同详情页
- `admin-system/ProjectManagement.vue`：项目管理列表页
- `admin-system/ProjectDetail.vue`：项目详情页
- `admin-system/FinanceManagement.vue`：财务管理页

### 3.2 资源管理页面
- `admin-system/EquipmentManagement.vue`：设备管理列表页
- `admin-system/EquipmentDetail.vue`：设备详情页
- `admin-system/KnowledgeBase.vue`：知识库管理页
- `admin-system/SupplierManagement.vue`：供应商管理列表页
- `admin-system/SupplierDetail.vue`：供应商详情页

### 3.3 数据中心页面
- `admin-system/DataStatistics.vue`：数据统计页
- `admin-system/ReportCenter.vue`：报表中心页
- `admin-system/LogManagement.vue`：日志管理页

### 3.4 系统配置页面
- `admin-system/RolePermission.vue`：角色权限管理页
- `admin-system/SystemConfig.vue`：系统参数配置页
- `admin-system/NotificationManagement.vue`：通知管理页

## 4. 开发原则
- 保持与现有系统风格一致
- 实现完整的CRUD操作
- 添加列表、详情、编辑功能
- 支持搜索、筛选、分页
- 使用Element Plus组件库
- 采用Vue 3 Composition API

## 5. 具体实现步骤
1. 添加路由配置
2. 创建页面文件
3. 实现基本布局和样式
4. 添加模拟数据
5. 实现交互功能
6. 优化用户体验

## 6. 预期效果
- 完善的中台功能页面
- 一致的视觉风格
- 流畅的用户体验
- 便于用户确认原型
