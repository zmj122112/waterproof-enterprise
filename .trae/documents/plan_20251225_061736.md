# Home.vue 视觉Bug修复计划

## Bug 1：首页头部 Logo 巨大化
### 问题分析
Header 里的 Logo 图片占满了整个屏幕，可能是因为缺少固定尺寸类名或被父容器撑开。

### 修复方案
1. 找到 Header 区域的 Logo 代码
2. 为 `<img>` 标签添加固定尺寸类名：w-10 h-10
3. 添加 object-contain 防止变形
4. 添加行内样式作为双重保险：style="width: 40px; height: 40px;"
5. 调整布局结构，增加副标题 "Since 1947"

### 修复代码位置
- 文件：src/views/c-mini/Home.vue
- 行号：125

## Bug 2：核心功能按钮布局与配色不协调
### 问题分析
按钮垂直排列，黄红配色过于刺眼，布局松散。

### 修复方案
1. 将容器布局改为 Grid 布局，一行两个按钮
2. 左侧按钮（咨询类）：白底红字 + 红边框（次要权重）
3. 右侧按钮（行动类）：红底白字 + 投影（主要权重）
4. 调整间距和尺寸，使其更加紧凑

### 修复代码位置
- 文件：src/views/c-mini/Home.vue
- 行号：166-177

## 修复文件列表
- src/views/c-mini/Home.vue

## 修复后效果
1. Logo 尺寸固定为 40px×40px，不会占满屏幕
2. 核心功能按钮采用双列等宽布局，配色主次分明，视觉效果更加专业
