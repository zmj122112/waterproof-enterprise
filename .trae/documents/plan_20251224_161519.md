# 图片修复与页面风格统一计划

## 一、问题分析

1. **工程案例图片显示问题**：
   - 第二张图片链接`https://images.unsplash.com/photo-1581094794329-cd119277ac36?w=400&q=80`返回404错误
   - 需要替换为有效的图片链接

2. **页面风格统一需求**：
   - 需要将当前良好的风格应用到所有其他页面
   - 确保颜色、布局、卡片样式等保持一致
   - 涉及页面：Appointment.vue、RepairEstimate.vue、WorkOrderProgress.vue、UserCenter.vue

## 二、修复方案

### 1. 图片链接修复

- 替换无效图片链接为有效的Unsplash图片
- 选择与防水相关的高质量图片
- 确保图片尺寸和比例统一

### 2. 页面风格统一

#### 统一规范
- **颜色系统**：使用深海蓝(#1a56db)作为主色，金橙色(#ff9500)作为辅助色
- **布局规范**：所有页面使用卡片式设计，圆角为12px，阴影为0 2px 8px rgba(0, 0, 0, 0.05)
- **间距规范**：模块间距为16px，卡片内边距为16px
- **字体规范**：标题使用18px，正文使用14px，辅助文字使用12px
- **图标规范**：使用Element Plus图标库，大小统一为20px

#### 页面修改计划
1. **Appointment.vue**：预约上门页面
2. **RepairEstimate.vue**：报修/估价页面
3. **WorkOrderProgress.vue**：工单进度页面
4. **UserCenter.vue**：个人中心页面

### 3. 实施步骤

1. 修复Home.vue中的图片链接
2. 定义统一的CSS类和布局规范
3. 逐个修改其他页面，确保风格统一
4. 验证所有页面的显示效果
5. 运行构建命令，确保没有错误

## 三、预期效果

1. 工程案例中的图片正常显示
2. 所有页面风格统一，包括颜色、布局、卡片样式等
3. 保持整体设计的专业性和美观性
4. 所有页面符合国企防水服务平台的定位
