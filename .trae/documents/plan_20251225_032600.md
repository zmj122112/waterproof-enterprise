# 结构性重构修复计划

## 全局 VI 修正
- **品牌红**：bg-red-600
- **品牌金（辅助色）**：bg-yellow-500

## 任务 1：彻底重写首页 Header (Home.vue)
### 问题分析
Header 布局混乱，搜索框和 Banner 叠放严重。

### 重构方案
- 使用标准文档流布局（Flex Column），避免绝对定位覆盖
- **容器 1 (红色顶栏)**：bg-red-600 pb-6 pt-safe rounded-b-[2rem] shadow-lg relative z-10
- **第一行**：Flex 布局
  - 左侧：Logo (w-8 h-8 rounded-full bg-white) + "月星防水"文字（白字加粗）
  - 右侧："客服"图标
- **第二行**：搜索框
  - bg-white/90 text-gray-500 rounded-full h-10 mx-4 mt-4 flex items-center justify-center

## 任务 2：消除页面巨大的空白间距 (Home.vue)
### 问题分析
Banner/服务优势与下方按钮之间有巨大空白。

### 修复方案
- 检查"服务优势"组件，移除固定高度，改用 py-3 实现自适应高度
- 检查"双按钮"容器，移除巨大 mt，改为 mt-4

## 任务 3：重设计"快速报修"与"预约上门"按钮 (Home.vue)
### 问题分析
一白一红像"选中/未选中"状态，易产生误解。

### 新方案
- 设计为"双主入口"风格，使用 VI 辅助色区分
- **布局**：grid grid-cols-2 gap-4 px-4
- **按钮 A (快速报修/估价)**：品牌金 (bg-yellow-500 text-white)，代表"金牌服务/快速"
- **按钮 B (预约上门)**：品牌红 (bg-red-600 text-white)，代表"官方/正式"
- **样式**：两个按钮均添加 rounded-xl、shadow-md，高度一致，文字居中

## 任务 4：重构"如何联系我们"卡片 (Appointment.vue)
### 问题分析
文字和图标垂直堆叠，浪费空间且丑陋。

### 重构方案
- 使用 Flex Row（左右布局）
- **容器**：bg-red-50 p-4 rounded-xl border border-red-100 flex justify-between items-center
- **左侧 (Info)**：flex flex-col gap-2，放置文字列表，文字左对齐
- **右侧 (QR)**：放置二维码图片，固定 w-24 h-24，下方加"扫码添加"小字
