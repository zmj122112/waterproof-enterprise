# C端小程序 - 个人中心原型设计

## 一、页面概述

### 1.1 页面定位
- 提供用户管理个人信息的入口
- 展示用户的历史工单和预约记录
- 提供服务评价和反馈功能
- 展示用户的积分和优惠券信息
- 提供系统设置和帮助中心入口

### 1.3 交互设计

#### 1.3.1 顶部导航栏
- 仅显示页面标题

#### 1.3.2 用户信息卡片
- **头像占位符**：
  - 点击头像可更换头像
  - 支持从相册选择或拍照
  - 动画效果：点击时的缩放+透明度变化效果
- **用户名/手机号**：显示用户的基本信息
- **编辑资料按钮**：
  - 点击进入编辑个人资料页面
  - 动画效果：点击时的缩放+颜色变化效果

#### 1.3.3 功能菜单
- **菜单列表项**：
  - 点击进入对应的功能页面
  - 右侧显示箭头图标，提示可点击
  - 动画效果：
    - 点击时的缩放+颜色变化效果
    - 进入下一页时的滑动动画
- **我的工单**：点击进入工单进度页面
- **我的预约**：点击进入预约记录页面
- **我的评价**：点击进入我的评价页面
- **我的积分**：点击进入积分明细页面
- **我的优惠券**：点击进入优惠券列表页面
- **地址管理**：点击进入地址管理页面

#### 1.3.4 系统设置
- **菜单列表项**：
  - 点击进入对应的设置页面
  - 右侧显示箭头图标，提示可点击
  - 动画效果：
    - 点击时的缩放+颜色变化效果
    - 进入下一页时的滑动动画
- **帮助中心**：点击进入帮助文档页面
- **意见反馈**：点击进入意见反馈页面
- **关于我们**：点击进入关于我们页面
- **设置**：点击进入系统设置页面

#### 1.3.5 退出登录按钮
- **退出登录**：
  - 点击弹出确认对话框，确认是否退出登录
  - 确认后清除用户登录状态，返回首页
  - 取消则关闭对话框
  - 动画效果：
    - 点击时的缩放+颜色变化效果
    - 弹出对话框时的淡入效果
    - 退出登录时的淡出效果

### 1.4 设计规范

#### 1.4.1 颜色方案
- 主色调：蓝色 (#1890ff) - 代表专业、信任
- 辅助色：橙色 (#ff7a45) - 代表活力、行动
- 背景色：浅灰色 (#f5f5f5)
- 文字色：深灰色 (#333333)、中灰色 (#666666)、浅灰色 (#999999)
- 边框色：淡灰色 (#e8e8e8)

#### 1.4.2 字体方案
- 标题：18px - 20px，粗体
- 正文：14px - 16px，常规
- 辅助文字：12px，常规

#### 1.4.3 图标方案
- 使用简洁、直观的线性图标
- 图标颜色与文字颜色保持一致
- 交互元素图标添加反馈效果

### 1.6 技术实现建议

#### 1.6.1 前端框架
- 使用微信小程序原生开发
- 或使用 Taro、uni-app 等跨端框架

#### 1.6.2 交互实现
- 使用微信小程序的 animation API 实现动画效果
- 使用 wx.chooseImage API 实现头像上传
- 使用 wx.navigateTo API 实现页面跳转
- 使用 wx.showModal API 实现确认对话框
- 使用 wx.setStorageSync/wx.getStorageSync API 实现登录状态管理

#### 1.6.3 数据来源
- 用户信息：调用后台用户服务 API
- 工单/预约/评价等数据：调用对应的后台服务 API
- 登录状态：本地存储
