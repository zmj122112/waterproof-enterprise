├── 4. 功能关联关系
│   ├── 4.1 依赖关系
│   │   ├── 工单管理 → 客户管理：工单创建需要客户信息
│   │   ├── 工单管理 → 物料管理：施工过程需要物料信息
│   │   ├── 师傅端工单接收 → 管理系统工单分配：只有分配的工单才能被接收
│   │   └── 积分计算服务 → 工单管理：根据工单完成情况计算积分
│   ├── 4.2 数据流向
│   │   ├── C端小程序预约 → 管理系统工单创建 → 师傅端工单接收
│   │   ├── B端批量报修 → 管理系统工单创建 → 师傅端工单接收
│   │   └── 师傅端施工记录 → 管理系统工单更新 → 积分计算服务
│   └── 4.3 调用关系
│       ├── 前端各模块 → 后端API：调用后端接口获取数据
│       ├── 后端业务模块 → 数据库：读写数据
│       └── 积分计算服务 → 工单数据：获取工单数据计算积分
