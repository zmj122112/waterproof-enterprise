# 管理系统 - 客户管理原型设计

## 一、页面概述

### 1.1 页面定位
- 为管理员提供防水工程客户的全生命周期管理功能
- 支持客户的创建、编辑、查询和统计
- 提供客户的批量操作和筛选功能
- 支持客户与工单、合同的关联管理
- 提供客户的分类管理和标签管理

### 1.3 交互设计

#### 1.3.1 顶部导航栏
- **系统logo**：点击返回系统首页
- **搜索框**：支持按客户名称、联系人、联系电话等关键词搜索
- **管理员中心**：点击进入管理员信息管理页面

#### 1.3.2 侧边导航栏
- **工单管理**：点击进入工单管理页面
- **客户管理**：当前页面，高亮显示
- **人员管理**：点击进入人员管理页面
- **物料管理**：点击进入物料管理页面
- **统计报表**：点击进入统计报表页面
- **系统设置**：点击进入系统设置页面

#### 1.3.3 操作按钮组
- **新增客户**：
  - 点击进入新增客户页面
  - 支持填写客户基本信息、联系人信息等
  - 动画效果：点击时的缩放+颜色变化效果
- **批量操作**：
  - 勾选多条客户后显示
  - 支持批量删除、批量冻结、批量导出等操作
  - 动画效果：点击时的缩放+颜色变化效果
- **导入客户**：
  - 点击弹出文件选择对话框，支持上传客户Excel文件
  - 支持批量导入多条客户记录
  - 动画效果：
    - 点击时弹出文件选择对话框
    - 上传过程中显示加载动画
    - 上传完成后显示成功/失败提示
- **导出客户**：
  - 点击导出客户列表为Excel文件
  - 支持导出全部或筛选后的客户
  - 动画效果：点击时显示导出进度提示

#### 1.3.4 筛选条件区
- **客户类型**：下拉选择客户类型，支持单选
- **客户状态**：下拉选择客户状态，支持单选
- **标签**：多选标签，用于筛选客户
- **搜索按钮**：点击根据筛选条件搜索客户
- 动画效果：
  - 下拉选择时的展开/收起动画
  - 多选框的勾选/取消勾选动画
  - 搜索按钮点击时的缩放效果

#### 1.3.5 客户列表
- **列表表头**：支持点击排序
- **复选框**：支持多选，用于批量操作
- **客户状态**：不同状态显示不同颜色
  - 正常：绿色 (#52c41a)
  - 冻结：灰色 (#d9d9d9)
- **标签**：显示客户的标签，支持点击查看相同标签的客户
- **操作列**：
  - **查看**：点击查看客户详情
  - **编辑**：点击进入编辑客户页面
  - **删除**：点击删除客户
  - 动画效果：
    - 操作按钮悬停时的颜色变化效果
    - 点击时的缩放效果

#### 1.3.6 客户详情弹窗
- **客户基本信息**：展示客户的核心信息
- **客户关联信息**：
  - 关联工单：展示与该客户相关的工单列表
  - 关联合同：展示与该客户相关的合同列表
  - 支持点击查看详情
  - 动画效果：列表项悬停时的背景色变化
- **操作按钮组**：
  - **编辑客户**：点击进入编辑客户页面
  - **删除客户**：点击删除客户，弹出确认对话框
  - **冻结客户**：点击冻结客户，改变客户状态
  - **关闭**：点击关闭弹窗
  - 动画效果：
    - 按钮点击时的缩放+颜色变化效果
    - 弹窗关闭时的淡出效果

### 1.4 设计规范

#### 1.4.1 颜色方案
- 主色调：蓝色 (#1890ff) - 代表专业、信任
- 辅助色：
  - 成功色：绿色 (#52c41a) - 代表正常状态
  - 警告色：黄色 (#faad14) - 代表待处理状态
  - 错误色：红色 (#ff4d4f) - 代表删除/冻结操作
  - 禁用色：灰色 (#d9d9d9) - 代表冻结状态
- 背景色：白色 (#ffffff)
- 文字色：深灰色 (#333333)、中灰色 (#666666)、浅灰色 (#999999)
- 边框色：淡灰色 (#e8e8e8)

#### 1.4.2 字体方案
- 标题：16px - 18px，粗体
- 正文：14px - 16px，常规
- 辅助文字：12px，常规
- 客户编号：14px，粗体，蓝色 (#1890ff)

#### 1.4.3 图标方案
- 使用简洁、直观的线性图标
- 图标颜色与文字颜色保持一致
- 交互元素图标添加反馈效果
- 状态图标：使用不同颜色区分不同状态

### 1.6 技术实现建议

#### 1.6.1 前端框架
- 使用React、Vue等现代化前端框架
- 采用响应式设计，适配不同屏幕尺寸

#### 1.6.2 交互实现
- 使用Ant Design、Element UI等UI组件库
- 使用Axios等库进行API请求
- 使用FileSaver.js等库处理文件下载
- 使用XLSX.js等库处理Excel文件导入
- 使用ECharts等库实现数据可视化

#### 1.6.3 数据来源
- 客户数据：调用后台客户管理服务API
- 工单数据：调用后台工单管理服务API
- 合同数据：调用后台合同管理服务API
- 文件上传/下载：调用后台文件服务API
