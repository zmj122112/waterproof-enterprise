### 1.1 页面定位

### 1.3 交互设计

#### 1.3.1 顶部导航栏
- **logo**：点击返回首页
- **搜索框**：点击进入搜索页面，支持搜索服务类型、常见问题等
- **我的**：点击进入个人中心页面

#### 1.3.2 轮播图区域
- **轮播图**：自动轮播，支持手动滑动切换
- **点击轮播图**：跳转至对应的活动详情页

#### 1.3.3 服务优势
- **服务优势卡片**：点击查看详细说明
- **动画效果**：悬停时轻微放大，显示详细文字说明

#### 1.3.4 核心服务
- **服务类型卡片**：点击进入对应的报修/估价流程
- **动画效果**：点击时缩放效果，跳转动画

#### 1.3.5 快捷入口
- **快速报修/估价**：点击进入报修/估价流程
- **预约上门**：点击进入预约上门流程
- **动画效果**：点击时变色+缩放效果

#### 1.3.6 服务套餐
- **套餐卡片**：点击查看套餐详情
- **预约按钮**：点击进入套餐预约流程
- **动画效果**：悬停时阴影加深，预约按钮变色

#### 1.3.7 底部导航栏
- **首页**：当前页面，高亮显示
- **估价**：点击进入智能报价计算器
- **我的**：点击进入个人中心
- **客服**：点击进入在线客服页面

### 1.4 设计规范

#### 1.4.1 颜色方案
- 主色调：蓝色 (#1890ff) - 代表专业、信任
- 辅助色：橙色 (#ff7a45) - 代表活力、优惠
- 背景色：浅灰色 (#f5f5f5)
- 文字色：深灰色 (#333333)、中灰色 (#666666)、浅灰色 (#999999)

#### 1.4.2 字体方案
- 标题：18px - 22px，粗体
- 正文：14px - 16px，常规
- 辅助文字：12px，常规

#### 1.4.3 图标方案
- 使用简洁、直观的线性图标
- 图标颜色与文字颜色保持一致
- 交互元素图标添加反馈效果

### 1.6 技术实现建议

#### 1.6.1 前端框架
- 使用微信小程序原生开发
- 或使用 Taro、uni-app 等跨端框架

#### 1.6.2 交互实现
- 使用微信小程序的 animation API 实现动画效果
- 使用 wx.navigateTo、wx.redirectTo 等 API 实现页面跳转
- 使用 wx.showToast、wx.showModal 等 API 实现反馈效果

#### 1.6.3 数据来源
- 轮播图数据：后台配置
- 服务优势和套餐：后台配置
- 优惠活动：后台配置

## 二、原型使用说明

### 2.1 如何查看原型
- 使用微信开发者工具打开原型文件夹
- 或使用在线原型工具（如 Axure、Figma）查看交互原型

### 2.2 如何进行交互测试
- 点击各功能按钮查看跳转效果
- 滑动轮播图查看轮播效果
- 悬停在卡片上查看动画效果

### 2.3 如何提供反馈
- 记录需要修改的页面或交互
- 标注具体的修改建议
- 提供参考示例（如有）
